# Park Circus Dream House Website

A clean, professional static website for Stephanie Stewart's interior design business.

## Website Structure

```
/
├── index.html           # Homepage with hero section and main content
├── about.html           # About page with profile and services
├── projects.html        # Projects overview with grid layout
├── contact.html         # Contact information and social links
├── css/
│   └── main.css         # Main stylesheet with responsive design
├── images/              # Image placeholders (ready for real photos)
└── projects/            # Individual project pages
    ├── grove-park.html
    ├── woodlands-terrace.html
    ├── stables-cottage.html
    ├── gilmartins.html
    ├── west-regent-street.html
    ├── pitt-street.html
    └── graham-terrace.html
```

## Design Features

- **Clean, Professional Aesthetic**: Inspired by the Park Circus Dream House Substack
- **Typography**: Georgia serif font for elegance and readability
- **Color Scheme**: Warm browns (#8B4513) with clean whites and grays
- **Responsive Design**: Mobile-first approach with tablet and desktop breakpoints
- **Navigation**: Sticky header with clear page indicators
- **Image Placeholders**: Ready for real project photos

## Content Included

- Complete business description and services
- All 7 project case studies with detailed information
- Client testimonials
- Contact information and social links
- External link to Substack blog
